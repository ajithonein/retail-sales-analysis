# Retail Sales Analysis

This project analyzes retail sales data to identify sales trends, top-performing products, customer purchasing behavior, and seasonal patterns.

## Objectives
- Analyze monthly and yearly sales trends
- Identify top-selling products
- Understand customer purchase behavior
- Generate business insights for decision making

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Peak sales periods identified
- High revenue product categories highlighted
- Customer buying trends analyzed
